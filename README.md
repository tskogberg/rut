# Rut

Rut is a CLI script that'll show you your routers uptime.

## Installation

Add this line to your application's Gemfile:

    gem 'rut'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rut

## Usage

    $ rut <router's ip address>

    Example:
  
    $ rut 10.0.1.1
    MyRouter has been up 102 days, 22:00:21.34
